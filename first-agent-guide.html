03 / The weekly loop

    
This is where it compounds.

    
Building the agent is the easy half. Managing it is what separates a toy from a system. Four moves, every week.

    

      
RUN
Run it Monday
Same job, same morning. Agents earn trust through rhythm, like people do.

      
READ
Read its work
All of it, like you'd read an intern's first report. Never skim, never auto-trust.

      
CORRECT
Correct the playbook
Never the output. One edit to the instructions kills that mistake forever.

      
WIDEN
Widen the job
Trust grows, scope grows. Complaints this month, pricing gaps the next.

    

    
and the corrections feed the next run. That's the compounding.

    

      The one rule people break
      
Fix the mistake by hand and you fix it once. Fix the playbook and you fix it forever. Thirty seconds of hand-fixing every week feels productive. It's how agents stay interns.

    

  






  

    
04 / Where first agents die

    
Three mistakes. All avoidable.

    

      

        
        
"Help me with market research" is a wish. "Top 3 complaints in my category, every Monday, in my sheet" is a job. The test: can you look at the output and say it did the job or it didn't? If you can't review it, you can't manage it.

      

      

        
        
An agent that doesn't know your customer, your price point, and what you've already ruled out will work hard on the wrong things. Same as a fresher with no handbook. Five minutes of pasted context beats fifty corrections later.

      

      

        
        
You wouldn't give a new hire ten responsibilities on day one. One job, run weekly, corrected in the playbook, until you trust it blind. Then widen. The teams running entire categories on agents all started with one boring job done well.
